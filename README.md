# 艺龙酒店爬虫Web平台
- 爬取站点：[http://ihotel.elong.com/region_179900/](http://ihotel.elong.com/region_179900/)
- 基于urllib库进行信息的爬取
- 基于Masonite框架进行Web平台的开发

这是一个Python爬虫课的大作业，非商业用途，仅供学习。

## 运行方式
配置`.env`文件

pip安装`requirements.txt`，控制台运行`craft`即可。

```
pip install -r requirements.txt # 安装外部库

craft install   # 安装依赖项

craft serve     # 启动服务器
```
